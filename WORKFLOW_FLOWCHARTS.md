## ğŸ¯ KEY API ENDPOINTS

### Email
- `POST /api/email/campaigns` - Create campaign
- `GET /api/email/campaigns` - List campaigns
- `POST /api/email/campaigns/[id]/send` - Send campaign
- `POST /api/webhooks/email` - Track events

### SMS
- `POST /api/sms/campaigns` - Create campaign
- `GET /api/sms/campaigns` - List campaigns
- `POST /api/webhooks/sms` - Incoming SMS
- `POST /api/webhooks/sms/status` - Delivery status

### Voice/IVR (Twilio)
- `GET /api/ivr/flows` - List IVR flows
- `POST /api/call-center/calls` - Initiate call
- `POST /api/webhooks/voice/status` - Call status
- `POST /api/call-center/ai/transcribe` - AI transcription

### AWS Connect
- `POST /api/call-center/aws-connect/make-call` - Start outbound call
- `POST /api/call-center/aws-connect/init` - Initialize call session
- `POST /api/call-center/aws-connect/end-call` - End and save call data
- `GET /api/call-center/aws-connect/agent-status` - Get agent availability
